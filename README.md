# SEKIN
Simplified interx and sekai deployment and setup

## Installation

1. Clone repo
```bash
https://github.com/MrLutik/sekin.github
```

2. Make bootstrap executable
```bash
cd sekin \
&& sudo chmod +x ./bootstrap.sh
```

3. Launch bootstrap
```bash
sudo ./bootstrap.sh
```

4. Build images
```bash
sudo docker compose build
```
